Log out Receipt

Инструкция по использованию:

Добавление рецепта:

  • Добавить текст рецепта в поле "Receipt"
  • Заполнить поля поля "Type", "Name", "Components" в разделе "Add receipt tags:" произвольным, понятным для Вас образом
  • Например: "салат", "оливье", "огурец"
  • Эти поля предназначены для последующего поиска сохраненного текста
  • В поле "Components" не обязательно писать все компоненты, например, салата, а только те, которые Вы хотите использовать при последующем поиске
  • Поле "Picture, URL" не обязательное, в него по желанию можно добавить URL подходящей картинки, скопировав его, к примеру, в интернете
  • При просмотре соответствующего рецепта будет показыватся также и эта картинка
  • Нажать кнопку "save"

Поиск рецепта:

  • Заполнить поля "By type", "By name", "By component", не обязательно все, произвольным образом
  • При нажатии кнопки "find" в поле "Search results" выводится список текстов, для которых значения, введенные в поля "By type", "By name", "By component" соответствуют сохраненным ранее в "Type", "Name", "Components" словам
  • При щелчке мыши на выбранной строке из "Search results" в поле Reseipt выведется соответствующий сохраненный текст
  • Если оставить пустыми все поля "By type", "By name", "By component", то в список "Search results" выведутся данные по всем сохраненным текстам
  • Если заполнить одно поле, то поиск будет производиться только по этому полю, если два - то по двум, если три - то по трем
  • Например, если в поле "By component" ввести слово ""огурец"", то получите список текстов, при сохранении которых в поле "Components" был указан огурец
  • В любое поле можно вводить только часть слова, а не все
  • Даже одну букву, но тогда выберется все, где есть эта буква. Например, если ввести "о", то получится поиск и по слову "Огурец", и по слову "помидОр", а также по другим, где есть эта буква

Удаление рецепта:

  • Выбрать строку в списке "Search results"
  • Нажать кнопку "delete" под списком
